# Excel-Schulung · Tag 2 — Unterlagen für Teilnehmer

Willkommen zu **Tag 2** des Kompaktkurses
*Excel & VBA – vom Rohdaten-Chaos zum Report auf Knopfdruck* (Fallbeispiel: Knoff-Hoff GmbH).

Tag 2 dreht sich um **VBA**: vom Makro-Recorder über Variablen, Schleifen und
eigene Funktionen bis zum Report-Generator – plus eine Vertiefung zu
**Funktionen & Variablen (ByVal/ByRef)** und eine **Debugging-Werkstatt**.

## Die Tag-2-Sets

7. **Recorder & Objektmodell** — Makro aufzeichnen, Code lesen, `Select` aufräumen
8. **Variablen & Datentypen** — `Dim`, `Option Explicit`
9. **Kontrollstrukturen** — `For`, `Do While`, `If`
10. **Prozeduren & Funktionen** — `Sub` vs. `Function`, Rückgabewerte
11. **Fehler & Debugging** — `On Error`, `Err`
12. **Finale: Report-Generator** (Meilenstein 2)
13. **Debugging-Werkstatt** — Bugs jagen mit dem VBA-Debugger (F8/F9/Direktfenster/Lokal/Watch) und Fehlerbehandlung (≈ 60 Min, Vertiefung zu Set 11)

## Debugging-Werkstatt starten (Set 13)

1. `Set-13_Debugging-Werkstatt_Aufgabe.xlsx` öffnen und als **`.xlsm`** speichern (Datei → Speichern unter → Typ *Excel-Arbeitsmappe mit Makros*).
2. VBA-Editor mit **Alt+F11** öffnen.
3. **Datei → Datei importieren** → `vba/Modul13_Debugging_BUGGY.bas` wählen.
4. Makro `ReportFuellen` mit **F8** Schritt für Schritt durchgehen und die Bugs jagen. Details auf dem *Anleitung*-Blatt und im PDF `aufgaben/Set-13_Debugging-Werkstatt_Aufgabe.pdf`.

## Foliensatz bedienen

`folien.html` im Browser öffnen, dann:

- **→ / Leertaste / Bild ab** — nächste Folie
- **← / Bild auf** — vorige Folie
- **Pos 1 / Ende** — erste / letzte Folie
- **N** — Trainer-Notizen ein-/ausblenden
